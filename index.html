Coordinator routing

Turns a vague request into a plan, routes tasks to specialists, and enforces constraints (latency budget, tool policies, and confidence thresholds).

What I optimize for

Not “agent vibes” — measurable production outcomes.

Metrics-first
  • Retrieval quality: precision/recall, reranking, failure modes.
  • Latency: orchestration overhead, queue fan-out, caching strategy.
  • Reliability: idempotency, retries, backpressure, timeouts.
  • Auditability: traces, tool logs, deterministic replay hooks.

How I build agents

A pragmatic pattern for shipping + evolving.

System design
  • Coordinator + specialists (retrieval, tools, validation).
  • Event-driven orchestration (SQS) with clear contracts.
  • State management (Redis/session store) + durable logs.
  • CI/CD for agent changes (tests, evals, rollout gates).
